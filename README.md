# Desafio Classificador de Nível de Herói - DIO

Projeto desenvolvido para consolidar os fundamentos de lógica de programação utilizando a linguagem JavaScript.

## Tecnologias Utilizadas
* JavaScript 
* Node.js (Ambiente de execução)
* Git & GitHub

## Conceitos Aplicados
* Declaração de variáveis e constantes
* Operadores lógicos e relacionais
* Estrutura de decisão condicional (`switch case` estruturado com lógica booleana)
* Laço de repetição (`while`) controlado por estado
* Manipulação de escopo de variáveis
